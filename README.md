# 포케크래프트 (PokeCraft) v16

마인크래프트 × 포켓몬 — 브라우저에서 돌아가는 복셀 샌드박스 게임.
외부 의존성 없음(three.js·peerjs 동봉). **친구와 온라인 멀티플레이 + 계정/백업 지원!**

## 👤 계정과 데이터 보관

- 타이틀에서 **로그인/가입**(이름+PIN) → 세이브가 계정별로 분리 보관 (가족·친구와 한 컴퓨터를 써도 안 섞여요)
- **내 세계**: 계정의 모든 세계 목록에서 골라서 플레이/삭제
- **백업 저장 / 백업 불러오기**: 계정 전체(모든 세계+포켓몬)를 파일 하나로 내보내고, 다른 컴퓨터·브라우저에서 그대로 복원
- 세이브에 버전이 기록되어 **게임이 업데이트돼도 옛 세이브가 자동 변환**돼요 (migrateSave)
- 주의: 세이브는 브라우저(localStorage)에 저장돼요 — "인터넷 사용 기록 삭제"로 사이트 데이터를 지우면 사라지니, 중요한 세계는 가끔 **백업 저장**을 눌러 파일로 보관하세요!
- 개발자 메모: 파일을 수정해 배포할 때는 index.html의 `?v=7`을 `?v=8`처럼 올려야 친구들 브라우저가 새 코드를 받아요

## 실행 방법

**방법 1 (가장 간단):** `index.html` 더블클릭 → 크롬에서 열기

**방법 2 (권장, 로컬 서버):**
```bash
cd pokecraft
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

**GitHub Pages 배포:** 저장소에 푸시 후 Settings → Pages → main 브랜치 선택. 멀티플레이까지 그대로 동작합니다.

## 🌐 멀티플레이 (친구와 같이!)

서버 없이 P2P(WebRTC)로 연결됩니다. 인터넷만 되면 같은 와이파이가 아니어도 OK.

1. **방장**: 타이틀 → 멀티플레이 → **방 만들기** → 6자리 코드를 친구에게 전달
2. **친구**: 타이틀 → 멀티플레이 → 코드 입력 → **참가하기**
3. 채팅은 **T** 키!

- 방장의 브라우저가 서버 역할 — 방장이 페이지를 닫으면 게임이 끝나요 (탭을 내려두는 건 OK)
- 세계는 방장 컴퓨터에 저장되고, 친구의 인벤토리/포켓몬은 친구 컴퓨터에 저장돼요
- 학교/회사 등 일부 보안이 강한 네트워크에서는 연결이 안 될 수 있어요

## 주요 기능

### 마인크래프트
- 절차 생성 무한 지형: 5개 바이옴, 동굴, 광맥, 바다 + **부드러운 음영(앰비언트 오클루전)**
- 블록 캐기/설치, 도구 5종(곡괭이·도끼·삽·검·**호미**) × 4등급, 내구도
- 제작대, 화로, **상자**(아이템 보관), 침대, 횃불, TNT
- **농사**: 호미로 경작지 → 밀 씨앗(풀 드롭) → 밀 → 빵 / 골분(뼈)으로 즉시 성장
- **낚시**: 낚싯대(막대+실)로 물고기·아이템·**잉어킹**까지!
- **활과 화살**: 우클릭 길게 눌러 차징 → 발사 (부싯돌은 자갈에서)
- 낮밤 사이클, 흐르는 물 애니메이션, 몹(돼지·소·양·닭·좀비·스켈레톤·크리퍼)

### 포켓몬 모드 — 1세대 151종 전부!
- **전국도감 1번 이상해씨부터 151번 뮤까지 전부 등장** (바이옴별 스폰, 희귀도 5단계)
- 스타터 3종 + 야생 포켓몬, **파트너 포켓몬이 뒤를 따라다님** (P에서 끄기 가능)
- 포켓볼 3등급 제작·투척 포획, 턴제 배틀(16개 타입 상성·급소·명중률)
- 레벨업·진화(망나뇽까지 3단 진화!) + **이상한 사탕**(낚시 희귀 보상)
- **전설**: 프리져(설원)·썬더(평원/산)·파이어(사막/산)·망나뇽(물) + **뮤츠**(밤 초희귀) + **뮤**(어디든, 극악)
- 잉어킹(낚시/물가) → Lv.20 갸라도스!

### 편의 기능
- **좌표 HUD**: 화면 좌상단에 X/Y/Z와 바라보는 방위 상시 표시
- **멀티 위치 공유**: 우상단에 친구의 방향 화살표(내 시선 기준)·거리·좌표 표시 — 흩어져도 찾아갈 수 있어요
- **📖 조합법 책 (B키)**: 모든 레시피를 카테고리별로 열람, 재료가 충분하면 초록색으로 표시
- **F1**: 게임 중 언제든 조작법 보기 / **황금 사과**(금괴8+사과): 체력 전체 회복

## 🤝 공동개발 가이드

```bash
git clone https://github.com/fakeminjun7321/pokecraft.git
cd pokecraft
python3 -m http.server 8000   # http://localhost:8000 에서 테스트
```

- **빌드 과정 없음** — 파일 수정하고 새로고침하면 끝. main에 푸시하면 몇 분 안에 Pages에 자동 반영돼요.
- **꼭 지킬 것**: JS/CSS를 수정해 푸시할 때는 `index.html`의 `?v=7`을 `?v=8`처럼 올려주세요 (브라우저 캐시 때문에 안 올리면 유저가 옛 코드를 봐요).
- 큰 기능은 브랜치 파서 작업 후 PR로 합치는 걸 추천: `git checkout -b feature/이름`
- 파일 지도: `world.js`(지형·청크·메싱) / `pokemon.js`(151종 데이터·배틀 — DEX 테이블에 종 추가) / `content.js`(블록·아이템·조합법) / `net.js`(멀티 동기화) / `player.js`(조작) / `ui.js`(인벤토리·메뉴) / `main.js`(루프·입력) / `account.js`(계정·세이브 버전)
- **세이브 형식을 바꿀 때는** `account.js`의 `SAVE_VERSION`을 올리고 `migrateSave`에 변환 코드를 추가해야 유저 세이브가 안 깨져요.
